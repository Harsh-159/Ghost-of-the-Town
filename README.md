## **About The Game**
A small game about a triangular ghost returning to spirit world after meeting his family at Halloween.



<br/>

### **Tech Stack used**

* [Unity](https://unity.com/)

<br/>

## **Key Features Include**
1. Movement
2. Story telling
3. Minimap
4. Switch to new level once old level is completed
